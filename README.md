lisping
=======

一个借鉴了clojure语法的命令式解释型语言

一个源文件就是一棵语法树.
调试时每个堆栈对应一棵语法树,
不仅可以控制进入, 返回函数,
还可以控制进入, 返回子表达式.
可以查看每个堆栈正在执行语法树的位置.
每执行一个表达式后, 可以看到表达式执行结果.

```clj
if (a > 10 
	&& a < 20 
	&& a % 2 == 1
	) {
	// pass
}

if (a > 10) {
	if (a < 20) {
		if (a % 2 == 1) {
			// pass
		}
	}
}

(if
	(&&
		(> a 10)
		(< a 20)
		(= (% a 2) 1)
nil
```
